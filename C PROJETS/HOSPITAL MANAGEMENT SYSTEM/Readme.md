# Hospital Management System

A comprehensive hospital management system built in C for managing patient records, doctor schedules, and medical appointments.

## Features

- **Patient Management**: Complete patient registration and medical records
- **Doctor Management**: Doctor profiles with specialization and availability
- **Appointment System**: Schedule, view, and manage medical appointments
- **Data Persistence**: File-based data storage
- **User-friendly Interface**: Menu-driven console interface

## Technologies Used

- **Language**: C
- **Compiler**: GCC

## Project Structure

# Hospital Management System

A comprehensive hospital management system built in C with modular architecture.

## Project Structure
- `main.c` - Program entry point
- `hospital.h` - Header file with structures and prototypes  
- `patient.c` - Patient management module
- `doctor.c` - Doctor management module
- `appointment.c` - Appointment scheduling module
- `utils.c` - Utility functions

# NOTE: **Compilation method**: run main.c
